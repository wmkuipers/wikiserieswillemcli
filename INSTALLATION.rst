============
Installation
============

At the command line::

    $ pip install wikiserieswillemcli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiserieswillemcli
    $ pip install wikiserieswillemcli

Or, if you are using pipenv::

    $ pipenv install wikiserieswillemcli
Or, if you are using pipx::

    $ pipx install wikiserieswillemcli
